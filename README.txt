Веб приложение представляет собой игру, по типу геогесера. В этой игре пользователь появляется в панораме со случайными координатами. Он должен
определить, где он находится. Пользователь с помощью панорамы ходит по местности, после он должен определить, где он находится.
После того, как пользователь введет координаты, приложение подсчитывает, баллы. Чем меньше разница в расстоянии между точкой подобранной программой
и точкой введенной пользователем, тем больше баллов.
Веб приложение предоставляет пользователю функционал регистрации, авторизирования. Баллы пользователей и количество матчей заносится в базу данных.
Пользователи могут видеть рейтинги, количество матчей других пользователй. Также они могут увидеть свой ретинг и количтество матчей на личной странице.
На личной страице также отображаются последние игры. Если пользователь - администратор, на личной странице появляется кнопка - войти в админку.
В админке аминистратор может редатировать и удалаять пользователей. Чтобы найти конкретного пользователя, в админке есть формы для пооиске.

Приложение разрабатывалось из расчета на теоритически возможное реальное использование. Для этого в странице рейтинг сделан переход между страницами.
Все пользователои не могут отображаться на одной странице, т.к. отображение большого количества пользователй может застопорить работу сайта.
Для тестирования в базе данных было создано около 50000 пользователей со случайными логинами, никами, паролями, количеством матчей.
Вполне реальная ситуация, на сайте одновременно находятся 30-50 пользователей, а зарегистрированно аккаунтов 50000 или больше. Это 
могут быть неактивные аккаунты, пользователь имеет несколько аккаунтов, и т.д. Конечно аккаунты можно удалять, например спуся год неактивности.
Но они сейравно могут накопиться. Их нужно правильно отобразить. Также вытаскивать из базы данных 50 тыс. пользователей для каждогозапроса
очень долго. Поэтому пользоваели сохраняются в оперативную память, и уже оттуда программа их получает. Все изменения с пользователями пишутся в базу
данных. Список пользователей внутри программы обновляется каждую минуту.
